\section{Introduction}
This guide explains how to replace the red placeholder values (\textcolor{red}{\textbf{XX.XX\%}}) 
with actual experimental results from the code execution.

\section{Step 1: Run the Code}
\begin{enumerate}
\item Install dependencies: \texttt{pip install -r requirements.txt}
\item Download the Kaggle dataset to \texttt{./data/}
\item Run training: \texttt{python main.py --mode train --dataset\_path ./data --output\_dir ./output}
\item Run evaluation: \texttt{python main.py --mode eval --model\_path ./output/best\_model.pth --dataset\_path ./data}
\item Generate visualizations: \texttt{python main.py --mode visualize --model\_path ./output/best\_model.pth --dataset\_path ./data}
\end{enumerate}

\section{Step 2: Extract Results}
After evaluation, check:
\begin{itemize}
\item Console output showing all metrics
\item \texttt{./results/tables/metrics\_main.tex} --- Main results table
\item \texttt{./results/tables/metrics\_robustness.tex} --- Robustness table
\item \texttt{./results/tables/metrics\_calibration.tex} --- Calibration table
\item \texttt{./results/tables/metrics\_ablation.tex} --- Ablation table
\item \texttt{./results/figures/} --- All PDF figures
\end{itemize}

\section{Step 3: Replace Placeholders}
For each \textcolor{red}{\textbf{XX.XX}} in the .tex files:
\begin{enumerate}
\item Find the corresponding value from the evaluation output
\item Replace the placeholder with the actual value
\item Remove the \textcolor{red}{} wrapper for final values (or keep for highlighting)
\item Remove the \% INS\'ERER VALEUR... comments
\end{enumerate}
